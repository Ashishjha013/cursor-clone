A **frontend UI recreation of the Cursor website landing page**, built using **HTML and CSS**.

This project focuses on replicating the **layout structure, typography, spacing system, and dark-theme design** of the original website while maintaining **clean semantic HTML and modern layout practices**.

The goal was to closely match the **visual hierarchy and modern SaaS landing page patterns** used in the Cursor website.

# 🧱 Sections Implemented

The following sections from the **Cursor landing page** were recreated.

---

## 1️⃣ Hero Section

Main landing section containing:

- Primary headline
- Supporting subheading
- Call-to-action button
- Editor preview image

Focus: **clear product messaging and strong visual hierarchy**.

---

## 2️⃣ Trusted By / Logo Strip

Displays company logos in a horizontal layout.

Design characteristics:

- Center-aligned structure
- Balanced spacing
- Subtle grayscale styling

Purpose: **build credibility and brand trust**.

---

## 3️⃣ Feature Sections

Highlights key product features such as:

- Code assistance
- Development automation
- AI-powered development tools

Layout design:

- Side-by-side text and image sections
- Alternating layout patterns
- Clear visual separation between sections

---

## 4️⃣ Code Preview Sections

Includes **dark-themed code blocks** representing the developer experience.

Features:

- Dark UI styling
- Clean code typography
- Highlighted preview sections

---

## 5️⃣ Testimonials Section

Displays user feedback cards.

Each card includes:

- Profile image
- Short testimonial quote
- Clean card UI

Purpose: **showcase user experience and product value**.

---

## 6️⃣ Changelog Section

Version cards displaying:

- Product updates
- Feature improvements
- Structured release notes

This layout reflects **modern product changelog design patterns**.

---

## 7️⃣ Call To Action Section

Encourages users to explore the product.

Includes:

- Clear CTA message
- Primary action button
- Center-aligned layout for focus

---

## 8️⃣ Footer

Multi-column footer layout including:

- Product links
- Company information
- Legal references
- Social links

Purpose: provide **clear navigation and additional resources**.

# 🔤 Typography

The project uses **Inter**, a modern UI font commonly used in SaaS products.

Primary font:

```
Inter, sans-serif
```

Fallback fonts:

```
system-ui, -apple-system, Segoe UI, Roboto, Arial
```

Font weights used:

- Regular (400)
- Medium (500)
- Semi-Bold (600)
- Bold (700)
